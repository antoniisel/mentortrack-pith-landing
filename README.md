# MentorTrack Pitch Landing

Сайт-презентация MentorTrack для защиты проекта.

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Открыть `http://localhost:5050`.

## Навигация

- кнопки в правом нижнем углу;
- клавиши `←` / `→`, `PageUp` / `PageDown` и пробел;
- вертикальный свайп на сенсорных экранах.

## Деплой

Проект подготовлен для бесплатного развёртывания на Render через `render.yaml`.
